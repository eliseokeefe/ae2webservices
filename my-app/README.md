# London Air Quality Tracker

This application utilizes a database in SQLite, a webpage created by Svelte, design done in tailwind css, and utilizing the London Air API (https://www.londonair.org.uk/Londonair/API/)

# Install 
1. Navigate to the folder my-app: cd my-ap
2. Install dependences: npm install 
3. Start the server: npm run dev 
4. Open browser and follow host link: http://localhost:5173

# Pages
- Home: landing page with login/register prompting, or a dashboard if the user is logged in already 
- Login/Register: use information to log in or create an account  
- Tracked Sites: view, filter, and edit tracked sites 
- Site Detail: view specific data for the borough selected 
- Site Search: search for specific boroughs in a drop down menu
